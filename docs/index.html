AI Prompts Library

A curated catalog of `.prompt.txt` files for engineers, maintainers, and cross-functional teammates. Copy the prompt, fill the placeholders, and ship faster without reinventing instructions.

View the repository on GitHub

How the Library Is Organized

  • Categories map to roles such as software-developer or maintainer.
  • Subdirectories reflect the job-to-be-done (for example, requirements, testing, authoring).
  • Files end in .prompt.txt and contain ready-to-run instructions you can paste into your AI assistant.

Contribute in Three Steps

  1. Draft your prompt following the blueprint and save it with the `.prompt.txt` suffix in the right directory.
  2. Run the style audit prompt to confirm structure and placeholder coverage.
  3. Open a pull request summarizing the scenario your prompt improves.

Roadmap

  • Automate lint checks for prompt metadata and formatting.
  • Publish quickstarts for integrating prompts into IDE snippets and CI templates.
